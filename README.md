# 🏌️✈️ 여행 안내문 생성기

골프 여행과 일반 여행을 위한 전문적인 안내문을 쉽고 빠르게 작성할 수 있는 웹 기반 시스템입니다.

## 🎯 프로젝트 개요

여행사나 단체 여행을 준비하는 분들을 위한 프리미엄 안내문 작성 도구입니다. 

**2가지 버전을 제공합니다:**
1. **⛳ 골프 여행 안내문** - 골프장 & TEE-UP 중심
2. **✈️ 일반 여행 안내문** - 일정표 중심

### 🧪 사용 방법

1. **입력 페이지** (`golf-advanced.html`)
   - 타이틀 입력 (필수)
   - 여행 기간 선택
   - 공항 미팅 / 현지 미팅 정보 입력
   - 골프장 추가 (무제한)
   - 일정 추가 (무제한)
   - "미리보기 & 공유하기" 클릭

2. **미리보기 페이지** (자동 이동)
   - 입력한 모든 정보 확인
   - 인쇄하기
   - 링크 복사 (다른 사람과 공유)
   - 수정하기 (입력 페이지로 돌아가기)

3. **저장 및 불러오기**
   - "현재 안내문 저장" - 전체 저장
   - "저장된 안내문 불러오기" - 불러오기
   - "현재 디자인을 템플릿으로 저장" - 디자인만 저장
   - "템플릿 불러오기" - 디자인만 적용

## ✨ 프리미엄 기능 (양쪽 모두 동일)

### 📸 이미지 시스템
- 타이틀 배경 이미지
- 공항/현지 미팅 이미지 (각각 별도)
- 항공편 정보 이미지
- 골프장별 개별 이미지
- 일정별 개별 이미지 (신규!)
- 회사 로고 추가
- 이미지 크기 조절 (50px ~ 800px)
- Base64 인코딩으로 저장

### 🎨 디자인 커스터마이징
- **8가지 한글 폰트 선택**
  - Pretendard (ExtraBold, Light, Medium, Regular)
  - 레시피코리아
  - SB 어그로 (B, L, M)
  - Noto Sans, 나눔고딕, 나눔명조 등
- **완전한 색상 커스터마이징**
  - 헤더 배경색
  - 타이틀 색상
  - 섹션별 배경색 (미팅, 항공편, TEE-UP, 일정, 숙소, 안내사항)
  - 테두리 색상
- **자동 여백 조절** (컨텐츠 양에 따라 최적화)

### 💾 저장 및 템플릿 시스템
- **다중 안내문 저장**
  - 여러 개의 안내문을 이름으로 구분하여 저장
  - 저장된 목록에서 선택하여 불러오기
  - localStorage 활용
- **템플릿 기능**
  - 현재 디자인 설정을 템플릿으로 저장
  - 템플릿 불러오기 (텍스트 유지, 디자인만 적용)
  - 템플릿 삭제 가능

  - 기본 정보: 여행제목, 출발일, 도착일
  - 공항 미팅: 장소, 날짜, 시간, 담당자, 전화 (5개)

  - 회사 정보: 회사명, 회사전화, 회사주소, 담당자명, 담당자전화, 담당자이메일 (6개)
- **3가지 파일 제공**:
  1. 빈 템플릿 다운로드 (`엑셀_템플릿_생성.html`)
  2. 예제 파일 다운로드 (`엑셀_예제파일_생성.html`) - 다낭 3박4일 실제 데이터
  3. 엑셀 파일 업로드 (`excel-import.html`)

## 🚀 사용 방법 (현재)

### 1. 데이터 입력 및 확인
1. `golf-advanced.html` 열기
2. 최소한 **타이틀** 입력
3. 원하는 정보 입력 (미팅, 골프장, 일정 등)
4. "미리보기 & 공유하기" 클릭

### 2. 데이터 확인 (개발자 도구)
1. F12 키로 개발자 도구 열기
2. Console 탭에서 다음 확인:
   - "수집된 데이터" 로그
   - airportMeeting 객체
   - localMeeting 객체
   - schedules 배열
3. localStorage 확인:
   - Application 탭 → Local Storage
   - `golfTripData` 키 확인

### 3. 저장/불러오기
1. "현재 안내문 저장" 버튼으로 저장
2. "저장된 안내문 불러오기"로 복원
3. "현재 디자인을 템플릿으로 저장"으로 스타일만 저장

### 4. 엑셀 자동 입력 (신규!)
1. **템플릿 다운로드**: `엑셀_템플릿_생성.html` → 빈 템플릿 받기
2. **예제 확인**: `엑셀_예제파일_생성.html` → 다낭 3박4일 예제 파일
3. **엑셀 작성**: 템플릿 형식에 맞춰 데이터 입력
4. **업로드**: `excel-import.html` → 엑셀 파일 드래그 또는 선택
5. **자동 변환**: 추출된 데이터 확인 후 "입력 폼 채우기" 클릭
6. **입력 완료**: golf-advanced.html로 자동 이동 및 데이터 적용
